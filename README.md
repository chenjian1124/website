# Vue3 项目

这是一个使用 Vue 3 + Vue Router + Ant Design Vue 构建的现代化前端项目。

## 技术栈

- **Vue 3.4.0** - 渐进式 JavaScript 框架
- **Vue Router 4.2.0** - 官方路由管理器
- **Ant Design Vue 3.2.20** - 企业级 UI 组件库
- **Vite 5.0.0** - 下一代前端构建工具

## 项目特性

- ✅ Vue 3 Composition API
- ✅ Vue Router 4 路由管理
- ✅ Ant Design Vue 组件库
- ✅ Vite 快速构建
- ✅ 响应式设计
- ✅ 中文注释

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 http://localhost:3000 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
├── src/
│   ├── views/          # 页面组件
│   │   ├── Home.vue    # 首页
│   │   └── About.vue   # 关于页面
│   ├── App.vue         # 根组件
│   └── main.js         # 入口文件
├── index.html          # HTML 模板
├── package.json        # 项目配置
├── vite.config.js      # Vite 配置
└── README.md           # 项目说明
```

## 开发说明

- 使用 Vue 3 的 `<script setup>` 语法
- 所有代码注释使用中文
- UI 组件使用 Ant Design Vue
- 路由配置在 `src/main.js` 中
- 页面组件放在 `src/views/` 目录下
